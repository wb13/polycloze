# polycloze

A tool for studying foreign language vocabulary using cloze tests.

[Screenshots](./screenshots)

## Usage

```bash
# Install everything needed to build front-end.
make init

# Run server.
make run

# Open in browser.
xdg-open http://localhost:3000
```
